# UK Road Safety Analysis: Accident Trends, Risk Factors & Safety Insights

A data-driven analytics project focused on understanding road accident patterns in the United Kingdom using large-scale historical accident data. The project combines Python-based ETL, exploratory analysis, statistical testing, and interactive Tableau dashboards to uncover accident hotspots, severity drivers, and actionable public safety recommendations.

---

## Project Overview

Despite strong traffic regulations, the UK continues to face a significant number of road accidents every year. Many of these incidents are preventable, but decision-makers often lack clear visibility into:

- Where severe accidents occur
- Why accident severity increases
- Which environmental or temporal factors raise risk
- How to prioritize targeted interventions

This project transforms raw accident records into strategic insights that can support smarter policy, infrastructure, and enforcement decisions.

---

## Problem Statement

Large volumes of road accident data exist, but they are often underutilized. Without structured analysis, authorities struggle to identify high-risk zones, severity patterns, and key causes behind serious or fatal accidents.

The objective of this project is to analyze historical UK road accident data and build an interactive decision-support system for road safety planning.

---

## Objectives

- Identify accident trends across years, time periods, and regions
- Analyze factors influencing accident severity
- Detect high-risk locations and scenarios
- Measure road safety KPIs
- Build executive dashboards for decision-makers
- Recommend targeted interventions based on evidence

---

## Dataset

**Source:** Kaggle - UK Road Safety Dataset  
**Coverage:** United Kingdom  
**Scale:** ~1.8 Million accident records  
**Columns:** 30+ variables  
**Period:** Multi-year historical dataset

Each row represents one reported road accident event.

### Key Fields

- Accident Index
- Accident Severity
- Number of Vehicles
- Number of Casualties
- Date
- Time
- Weather Conditions
- Light Conditions
- Road Type
- Urban / Rural Area
- Speed Limit
- Local Authority
